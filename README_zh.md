# Filmarks 电影评论采集器 (Filmarks Movie Comments Scraper)

[Back to Japanese Document (返回日文主文档)](./README.md)

---

一个基于 requests 和 BeautifulSoup 构建的轻量级网页结构化解析脚本。专门用于精准采集日本电影评论网站 Filmarks 特定作品下的用户评论数据。

## 核心特性
* 最新前端适配：已完美适配 Filmarks 官方近期更新的 c2- 前缀响应式前端类名架构。
* 长评自动展开 (N+1 容错)：当遇到需要点击展开的评论（続きを読む）时，脚本会自动提取该评论的独立详情页链接，发起次级请求补全完整文本。
* 防数据丢失机制：内置 flush() 磁盘同步，每抓取完一页数据即刻物理写入 CSV，防止意外中断造成数据丢失。

## 抗风控等待时间限制说明

为避免高频请求触发 Filmarks 服务器的安全防火墙（WAF）拦截，脚本设置了严格的策略性等待时间限制：

1. 主干翻页延迟 (主循环)：time.sleep(3)
每次下载完一个列表页后，强制休眠 3.0秒。规避高频并发造成的 IP 封禁风险。

2. 详情长评延迟 (子请求)：time.sleep(1.5)
当需要进入用户独立页面展开长评正文时，发起子请求前强制休眠 1.5秒。防止由于详情页频控导致抓取中断。

运行提示：如果单页中长评数量过多，脚本可能会在短时间内表现出暂停（卡住）状态，此为安全休眠策略正在执行，属于正常现象，请勿强行终止。

## 运行环境与依赖
* Python 3.7+
* 依赖库：requests, beautifulsoup4, lxml

安装依赖命令：
pip install requests beautifulsoup4 lxml

## 使用指南
1. 打开项目中的 main.py。
2. 修改全局常量 URL，替换为你需要采集的 Filmarks 电影首页 URL（例如：https://filmarks.com/movies/70258 ）。
3. 启动脚本：
python main.py

## 产出结果说明
运行成功后，会在根目录下生成 comments_filmarks.csv 文件。使用带 BOM 的 UTF-8 编码，确保在 Microsoft Excel 中直接打开时绝对不乱码。

* ID: 评论者的用户昵称（已自动剔除系统后缀）。
* Time: 评论发布时间（示例: 2026/05/14 06:02）。
* star: 评分（未打分用户显示 - ）。
* comments: 结构化清洗后的评论正文（已自动替换多余换行符并拼接长评全文）。
