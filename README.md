# Filmarks Movie Comments Scraper (Filmarks 映画レビュークローラー)

[中文] 一个基于 `requests` 和 `BeautifulSoup` 构建的网页结构化解析脚本。专门用于精准采集日本电影评论网站 Filmarks 特定作品下的用户评论数据。
[日本語] `requests` と `BeautifulSoup` をベースに構築された軽量なスクレイピングスクリプトです。日本の映画レビューサイト「Filmarks」から特定の作品のユーザーレビューデータを正確に収集します。

---

## 核心特性 / 主な機能

- **最新版適配 / 最新UI対応**
  - [中文] 已完美适配 Filmarks 官方近期更新的 `c2-` 前缀前端类名组件架构。
  - [日本語] Filmarksが最近更新した `c2-` プレフィックスのレスポンシブDOMクラス名に完全対応しています。

- **长评自动展开 / 長文レビューの自動取得**
  - [中文] 当遇到需要点击展开的评论（`続きを読む`）时，脚本会自动提取详情页链接，发起次级请求补全完整文本。
  - [日本語] 「続きを読む」が含まれる省略されたレビューに遭遇した場合、スクリプトは詳細ページのリンクを自動的に抽出し、サブリクエストを送信して全文を補完します。

- **防数据丢失机制 / データ紛失防止メカニズム**
  - [中文] 内置 `flush()` 磁盘同步，每抓取完一页数据即刻物理写入 CSV，防止意外中断造成数据丢失。
  - [日本語] `flush()` によるディスク同期処理が組み込まれており、1ページ取得するごとにCSVへ強制書き込みを行うため、途中で強制終了してもデータが失われません。

---

## 抗风控等待时间限制说明 / アクセス制限・ウェイトコントロール

[中文] 为避免高频请求触发 Filmarks 服务器的安全防火墙（WAF）拦截，脚本设置了严格的策略性等待时间限制：
[日本語] 高頻度のリクエストによるFilmarksサーバーのファイアウォール（WAF）制限を回避するため、スクリプトには厳格な待機時間（ウェイト）が設定されています：

1. **主干翻页延迟 / ページ間ウェイト (主ループ)**: `time.sleep(3)`
   - [中文] 每次下载完一个列表页后，强制休眠 **3.0秒**。
   - [日本語] 各レビュー一覧ページをダウンロードした後、一律 **3.0秒** のスリープを強制します。
2. **详情长评延迟 / 詳細ページウェイト (サブリクエスト)**: `time.sleep(1.5)`
   - [中文] 当需要进入用户独立页面展开长评正文时，发起子请求前强制休眠 **1.5秒**。
   - [日本語] 「続きを読む」の全文を取得するために個別ページにアクセスする際、アクセス前に **1.5秒** のスリープを強制します。

*⚠️ [警告/警告] 如果单页中长评数量过多，脚本可能会在短时间内表现出暂停（卡住）状态，此为安全休眠策略正在执行，属于正常现象，请勿强行终止。*
*⚠️ 1ページ内に長文レビューが多数存在する場合、一時的にスクリプトが停止（フリーズ）したように見えることがありますが、これは安全待機戦略が実行されているためです。異常ではありませんので終了しないでください。*

## 产出结果说明 / 出力データについて

[中文] 运行成功后，会在根目录下生成 `comments_filmarks.csv` 文件。使用带 BOM 的 UTF-8 编码，确保在 Microsoft Excel 中直接打开时**日语不乱码**。
[日本語] 実行が完了すると、ルートディレクトリに `comments_filmarks.csv` が生成されます。BOM付きUTF-8エンコードを採用しているため、Microsoft Excelで直接開いても**日本語の文字化けが発生しません**。

### 数据矩阵字段 / データフィールド:
- **`ID`**: [中文] 评论者的用户昵称 / [日本語] レビュー投稿者のユーザー名
- **`Time`**: [中文] 评论发布时间（示例: `2026/05/14 06:02`）/ [日本語] レビュー投稿時間
- **`star`**: [中文] 评分（未打分用户显示 `-`）/ [日本語] ユーザーのスコア（未評価の場合は `-`）
- **`comments`**: [中文] 结构化清洗后的评论正文（已自动替换多余换行符并拼接长评全文）/ [日本語] 構造化・クレンジング済みのレビュー本文（改行コードの除去および長文の復元が完了しています）
